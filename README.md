## 👨‍💻 About

This is the source code for my personal portfolio website — built to showcase my work, skills, and experience as an **Associate Engineer (Software)** at **Rockwell Automation**, Pune.

The site is designed to be fast, interactive, and fully responsive — with no frameworks, just clean **HTML, CSS and vanilla JavaScript**.

---

## ✨ Features

- **Animated loader** — cycling status messages with a progress bar
- **Custom cursor** with a trailing ring and hover effects
- **Interactive particle canvas** — connects to your cursor on hover
- **Typing animation** — cycles through roles dynamically
- **3D card tilt** — on all skill and project cards
- **Animated skill bars** — triggered on scroll
- **Auto-scrolling certifications ticker** — pauses on hover
- **Scroll reveal animations** — smooth entrance for every section
- **Fully responsive** — works on mobile, tablet and desktop
- **No frameworks** — pure HTML/CSS/JS, zero dependencies
